# HDU-Computer-Art-Design

> **杭州电子科技大学 (HDU) - 计算机辅助图形设计课程项目仓库**

本项目用于存放 2023 学年《计算机辅助图形设计》课程的相关实验、作业及期末设计作品。涵盖了从基础图像处理到复杂平面排版的设计全过程。

---

## ✍️笔者总结
计算机辅助设计是工业设计专业大一下的必修课，整个课程期间会使用PS，AI等Adobe工具仿照上课要求手搓示例图，期末考试也是给出一张图片在ps中进行仿照实现，最后给出课程分数。这节课是一门大课一节课包含四节小课，加上教室里面插座数量有点少，建议游戏本用户可以早点去抢占插座位，不然可能会发生上课到一半电脑突然没电的情况（不排除小登换教室了）。笔者当时刚大一这节课是早八，有幸解锁了上课时靠窗打只狼的成就，也不失为一段佳话。

## 🛠 开发与设计环境

* **图像处理**: Adobe Photoshop (主要用于 `.psd` 文件处理)
* **矢量设计**: Adobe Illustrator (主要用于 `.ai` 文件处理)
* **版本管理**: Git + [Git LFS](https://git-lfs.github.com/) (用于管理大型二进制设计文件)

## 📝 核心项目介绍

### 1. 甜品宣传单设计
该项目是课程中的重点练习。通过对图层样式、字体排版以及色彩心理学的应用，完成了一套可供印刷的甜品促销海报。

### 2. 期末结课作品
汇总了学期内掌握的所有 CAD 设计技巧，展示了在计算机辅助下的艺术创作能力。
