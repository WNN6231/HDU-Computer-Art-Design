# HDU-Computer-Art-Design

> **杭州电子科技大学 (HDU) - 计算机辅助设计课程项目仓库**

本项目用于存放 2023 学年《计算机辅助设计》课程的相关实验、作业及期末设计作品。涵盖了从基础图像处理到复杂平面排版的设计全过程。

---

## 📁 目录结构

| 文件夹 / 文件 | 内容说明 | 备注 |
| :--- | :--- | :--- |
| **实例04(1)** | 课堂同步练习实例 | 基础操作与功能演练 |
| **期末考试** | 课程结课大作业 | 包含完整的设计方案与成品 |
| **甜品宣传单设计** | 专项商业排版练习 | 已启用 Git LFS 托管大型源文件 |
| **素材 / 素材(1)** | 设计所需原始资源 | 包含高清图库及临摹参考图 |
| **.gitattributes** | Git 配置文件 | 用于 LFS 追踪 `.psd` 等大文件 |
| **wnn.psd / wnn_camera.psd** | Photoshop 设计稿 | 包含完整图层、蒙版及调色记录 |
| **wnn.ai / wnn(7).ai** | Illustrator 矢量稿 | 包含路径、锚点及排版矢量数据 |
| **仙人掌_画板 1.jpg** | 作品渲染/导出图 | 最终视觉效果的快速预览 |

---

## 🛠 开发与设计环境

* **图像处理**: Adobe Photoshop (主要用于 `.psd` 文件处理)
* **矢量设计**: Adobe Illustrator (主要用于 `.ai` 文件处理)
* **版本管理**: Git + [Git LFS](https://git-lfs.github.com/) (用于管理大型二进制设计文件)

## 📝 核心项目介绍

### 1. 甜品宣传单设计
该项目是课程中的重点练习。通过对图层样式、字体排版以及色彩心理学的应用，完成了一套可供印刷的甜品促销海报。

### 2. 期末结课作品
汇总了学期内掌握的所有 CAD 设计技巧，展示了在计算机辅助下的艺术创作能力。
